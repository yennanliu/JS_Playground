## JS_Playground

CRUD methods | HTTP Method	Definition 
------------ | -------------
Create	|  POST	Creates a new resource
Read	|  GET Retrieves a resource
Update	|  PUT/PATCH Updates an existing resource
Delete	|  DELETE Deletes a resource
